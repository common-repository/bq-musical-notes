Converts musical notes that use 'b' for flat and '#' for sharp into professional looking notes.
Provides a shortcode to insert chord diagrams for stringed instruments into posts and pages.


== Description ==

Automatically converts musical notes written as Eb or F#, into notes written with real flat and sharp characters.
Works on all posts, pages and comments as well as titles. Does not convert feeds. Can be turned on and off from inside posts and pages.
Provides a shortcode to insert chord diagrams for stringed instruments with any number of strings.


== Installation ==

1. Install the plugin from wordpress.org, or unzip bqnotes.zip or bqnotes.tar.gz into your plugins directory.
2. Activate the BQ Musical Notes plugin through the Plugins menu in WordPress.
